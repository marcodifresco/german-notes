# Conjugation Perfekt
For [[Verbs]] in one place:
[[Pronouns]] + [[Haben]] + **ge**VERB

For [[Verbs]] that indicate movement to another place:
[[Pronouns]] + [[Sein]] + **ge**VERB

For [[Typ - Modal verben]]:
Prefix-verb --> [[Pronouns]] + [[Haben]]/[[Sein]] + prefix + **ge** + verb konjugation
